# スペランカーMOD
かつてファミコンでその名を馳せた偉大なる先人「スペランカー」の類い希なる能力を与えるMODです。

## Distribution and Usage
https://tsuteto.hacca.jp/mcmods

## Dependencies
- Minecraft 1.7.10
- Forge 10.13.4.1230
- Player API
